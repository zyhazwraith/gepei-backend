# 部署与交付检查清单 (Deployment Checklist)

本文档记录了项目上线/交付时，需要甲方（客户）准备的账号、密钥及配置项。

## 1. 第三方服务账号与密钥

### 1.1 腾讯地图 (Tencent Map)
- **用途**: 用于前端地图展示、定位及逆地址解析。
- **需要提供**: `Map Key` (Web端)
- **获取方式**:
    1. 访问 [腾讯位置服务官网](https://lb.qq.com/) 注册企业/个人账号并实名认证。
    2. 控制台 -> 应用管理 -> 创建应用 -> 添加 Key。
    3. **安全配置**: 启用“域名白名单”，填入正式上线域名的地址（如 `www.example.com`）。
- **配置位置**: 生产环境 `.env` 文件中的 `VITE_TENCENT_MAP_KEY`。
