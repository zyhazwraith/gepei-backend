## 🛠️ 开发环境 (Development)

### HTTPS 启用
前端服务 (`client`) 默认启用 HTTPS (`https://localhost:5173`)，主要为了支持 Geolocation (LBS) 等需要安全上下文的 API。

- **证书说明**: 使用 Vite 自动生成的自签名证书。
- **浏览器警告**: 首次访问时，浏览器会提示"您的连接不是私密连接"，这是正常现象。请点击"高级" -> "继续前往 localhost (不安全)"。
- **配置**: 由 `@vitejs/plugin-basic-ssl` 插件支持。
