```tsx
// 表单字段间距

```

## 3. 个人中心对比

### 设计稿尺寸（05_profile.png）
- **顶部橙色区域高度**: 200px
- **头像大小**: 80px
- **"认证为地陪"按钮**: 宽度120px，高度36px
- **菜单卡片间距**: 16px
- **单个菜单项高度**: 48px（图标+文字+箭头）
- **图标大小**: 24px（无背景圆形）
- **卡片内边距**: 16px

### 当前实现问题
- ❌ **菜单卡片间距**: mb-4（16px）→ 设计稿也是16px，但视觉上看起来过大
- ❌ **菜单项padding**: py-3.5（14px）→ 应该是12px左右
- ❌ **图标容器**: w-10 h-10（40px）带背景圆形 → 设计稿是24px无背景
- ⚠️ **设计稿问题**: 图标直接显示，没有背景圆形，但这样不够美观

### 修复方案
```tsx
// 菜单卡片间距（保持16px，但调整卡片圆角和阴影）
mb-4  // 保持

// 菜单项padding
py-2.5  // 从py-3.5改为py-2.5（10px）

// 图标容器（保留背景圆形，但缩小）
w-8 h-8  // 从w-10 h-10改为w-8 h-8（32px）
```

### 🤔 设计建议
**设计稿中图标没有背景圆形，但我建议保留背景圆形**，理由：
1. 视觉层次更清晰
2. 符合现代移动应用设计规范
3. 与首页的图标风格一致
