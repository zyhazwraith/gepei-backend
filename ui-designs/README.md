## 📐 设计规范

### 颜色方案
- 主色调：珊瑚橙 (#FF6B6B)
- 背景色：白色/浅灰
- 文字色：深灰 (#333333)

### 页面结构
- 顶部导航栏：固定高度，包含标题和功能按钮
- 底部Tab栏：5个主要功能入口
  - 首页
  - 地陪
  - 消息
  - 高端定制
  - 我的

### 交互说明
- 所有表单页面支持输入验证
- 列表页面支持下拉刷新
- 支付流程需要二次确认
